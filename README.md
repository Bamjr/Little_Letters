# Little Letters ✉️

Little Letters is a simple web application for browsing and collecting
quotes from around the internet. The website displays random quotes in a
clean card-based layout, allowing users to explore and read them in a
visually pleasant interface.

Users can also add their own quotes through a form with image upload
preview. This project focuses on frontend structure, layout design, and
basic JavaScript functionality such as fetching data from an API and
handling form input.

## Live Demo

https://bamjr.github.io/Little_Letters/

## Features

-   Display random quotes from an external API
-   Clickable quote cards with a full quote view page
-   Add your own quote (demo submission)
-   Basic demo login system (frontend only)
-   Responsive layout for different screen sizes
